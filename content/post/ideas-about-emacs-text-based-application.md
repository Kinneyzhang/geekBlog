# mode 与 application
mode 是 emacs 的核心概念之一，分类两类：major mode 和 minor mode

emacs 中的插件大多数都是 mode 类的，application 类的很少。这是符合直觉的，因为 emacs 的本质是编辑器，所有大多数的插件都是在为更好编辑做拓展，无论是 显示优化(主题)、文本搜索定位、文本/代码补全
